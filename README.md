# Continuum: The Vanguard Suite
### Unified Clinical Intelligence & Personal Health Orchestration

Continuum is an enterprise-grade personal health intelligence platform designed to bridge the chasm between care seekers and medical specialists. By leveraging reactive clinical synchronization, generative AI orchestration, and a high-trust security architecture, Continuum transforms fragmented medical data into a longitudinal, actionable health narrative.

---

## 🏗️ Platform Architecture

Continuum operates as a dual-portal ecosystem, synchronized by a reactive intelligence bridge.

### 1. Care Seeker Portal (The Sanctuary)
A cinematic dashboard designed for high-agency health management.
- **🧠 Health Memory**: A natural-language interface for logging symptoms, medications, and clinical events, instantly structured by Gemini AI.
- **🩺 Symptom Intelligence**: Proactive pattern analysis and longitudinal symptom mapping.
- **📑 Financial Explainer**: AI-powered medical bill OCR and itemized cost decoding.
- **👨‍👩‍👧‍👦 Care Circle (Guardians)**: Multi-profile orchestration for family health delegation.
- **🆔 Emergency Passport**: A high-trust SOS interface with reactive clinical record mirroring.

### 2. Specialist Portal (Banguard Suite)
A professional cockpit for medical specialists and clinical teams.
- **📊 Clinical Cockpit**: Longitudinal patient views with real-time consultation mapping.
- **💊 Staged Prescription Engine**: A high-fidelity workflow for issuing multi-medication clinical orders.
- **📂 EHR Synchronization**: Intelligent indexing of medical reports and clinical summaries.
- **💬 Direct Care Bridge**: Secure, context-aware consultation channels with patients.

## 🔐 Intelligence & Security Infrastructure

### 🎖️ Tiered Subscription Architecture
Continuum's feature set is governed by an intelligent gating layer:
- **Free**: Core Health Memory and personal logging.
- **Trial**: Full access to the Care Circle (Guardians) and Advanced Analytics.
- **Premium**: Infinite Health Memory, SOS Sync, and Priority AI processing.
- **Institutional**: Enterprise EHR connectivity and multi-specialist team management.

### 🛡️ Security & Privacy
- **Supabase RLS**: Direct database-level Row Level Security ensures absolute data isolation.
- **Audit-Ready Logging**: Comprehensive tracking of clinical modifications and access.
- **Vault Encryption**: Secure storage for high-sensitivity medical reports and prescriptions.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Supabase Project URL & Anon Key
- Google AI API Key (Gemini)

### Installation
1. **Clone & Install**
   ```bash
   git clone https://github.com/batmanven/continuum.git
   npm install
   ```

2. **Environment Configuration**
   Create a `.env` file with your Vanguard credentials:
   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   VITE_GOOGLE_AI_API_KEY=your_gemini_key
   ```

3. **Database Stabilization**
   Apply the latest clinical migrations:
   ```bash
   supabase db push
   ```

4. **Launch Platform**
   ```bash
   npm run dev
   ```
